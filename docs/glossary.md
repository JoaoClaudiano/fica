# Glossário do FICA

Este glossário explica os principais termos utilizados no projeto, para uniformizar compreensão entre equipe, usuários e avaliadores.

---

## Termos-chave

**Aluno**  
Estudante participante do sistema, cuja frequência e bem-estar são monitorados de forma ética.

**Professor**  
Profissional responsável pelo acompanhamento, análise e intervenções preventivas.

**Dashboard**  
Interface de visualização e controle de dados por usuário (aluno ou professor).

**Check-in de bem-estar**  
Registro voluntário de estado emocional ou social do aluno.

**Índice de Risco de Evasão (IRE)**  
Indicador calculado com base em frequência, desempenho e bem-estar.  
**Sempre explicável e contestável.**

**Alerta**  
Sinal sugerindo atenção para um aluno ou grupo, **não punitivo**.

**Gamificação**  
Sistema de metas e conquistas que motiva alunos sem estigmatizar.

**Empty State**  
Tela ou componente exibido quando não há dados disponíveis, garantindo experiência consistente.

**Mobile First**  
Princípio de design priorizando dispositivos móveis antes de desktop.

**Acessibilidade**  
Conjunto de práticas que permitem que pessoas com diferentes habilidades usem o sistema sem barreiras.
