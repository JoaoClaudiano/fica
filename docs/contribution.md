# Como Contribuir com o FICA

O FICA é um projeto open source, criado para impactar positivamente a educação e o bem-estar escolar.  
Sua contribuição é bem-vinda, seja código, design, documentação ou sugestões.

---

## Regras Básicas

1. **Fork e clone** o repositório.
2. Crie uma **branch** específica para sua contribuição:
   ```bash
   git checkout -b minha-contribuicao
   
3.	Faça suas alterações, seguindo os padrões de código e estilos existentes.

4.	Abra um Pull Request descrevendo:
	•	Objetivo da mudança
	•	Como testar
	•	Qual impacto para o projeto
	5.	Aguarde a revisão e feedback.

---

## Áreas onde você pode contribuir
	•	Código frontend (HTML, CSS, JS)
	•	Componentes reutilizáveis
	•	Documentação
	•	Testes e validação de UX
	•	Conteúdo institucional
	•	Ícones, ilustrações e recursos gráficos
	•	Propostas de novas métricas e indicadores

---

## Código Ético
	•	Toda contribuição deve respeitar ética, privacidade e bem-estar dos alunos.
	•	Evite coleta ou compartilhamento de dados pessoais sem consentimento.
	•	Priorize acessibilidade e inclusão
